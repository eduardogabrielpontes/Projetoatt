Instruções para rodar o projeto no XAMPP (Windows) / MAMP (Mac) / LAMP:

1) Copie a pasta 'Projetoatt' inteira para a pasta 'htdocs' do seu XAMPP.
   Exemplo: C:\xampp\htdocs\Projetoatt

2) Inicie o Apache no XAMPP Control Panel.

3) Abra o navegador e acesse:
   http://localhost/Projetoatt/index.php

4) Clique em 'Gerenciar Clientes' para usar o CRUD.

Arquivos incluidos:
- index.php
- /clientes/listar.php
- /clientes/cadastrar.php
- /clientes/editar.php
- /clientes/excluir.php
- /clientes/clientes.json (com exemplos)
